# Git Course Practice Repository

## 📌 About the Project
This repository is created to learn and practice Git and GitHub concepts using VS Code and Git Bash.  
It includes basic Git operations such as initializing a repository, committing changes, working with branches, and pushing code to GitHub.

---

## 🚀 Features
- Git initialization and configuration
- Staging and committing files
- Branch creation and deletion
- Linking local repository with GitHub
- Using SSH for secure GitHub access
